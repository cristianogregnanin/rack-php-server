# rack-php-server

Sometimes I need to serve php files but I don't want to install Apache or similar heavy web server.

## Installation

Clone the repo.

Execute:

    $ bundle
    
In the repo you can find a simple index.html file, free feel to replace it with your index.php 

## usage

Execute:

    $ rackup
